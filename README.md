<pre># User Management REST API

A simple RESTful API built using Flask that performs CRUD (Create, Read, Update, Delete) operations on user data. The API stores user information in an in-memory dictionary and provides endpoints for managing users through standard HTTP methods.

## Features

* Get all users (GET)
* Get a specific user by ID (GET)
* Add a new user (POST)
* Update existing user information (PUT)
* Delete a user (DELETE)
* JSON-based request and response handling
* Lightweight and beginner-friendly Flask implementation

## Technologies Used

* Python
* Flask
* REST API
* HTTP Methods (GET, POST, PUT, DELETE)
* Postman / Curl for API testing

## API Endpoints

| Method | Endpoint    | Description             |
| ------ | ----------- | ----------------------- |
| GET    | /users      | Retrieve all users      |
| GET    | /users/<id> | Retrieve a user by ID   |
| POST   | /users      | Add a new user          |
| PUT    | /users/<id> | Update an existing user |
| DELETE | /users/<id> | Delete a user           |

## Learning Outcome

This project demonstrates the fundamentals of REST API development using Flask, including route creation, request handling, JSON processing, and CRUD operations.
